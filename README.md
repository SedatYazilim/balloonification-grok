# Balloonification - Grok Edition

A playful memecoin website powered by Grok Intelligence.

## Features
- Responsive design with Grok-inspired dark theme
- Interactive balloon squad showcase
- Comprehensive Grok AI capabilities section
- Evolution timeline of Grok versions
- Grok modes and personas showcase

## Live Demo
Visit the live site at: https://balloonification-grok.vercel.app

## Getting Started
1. Clone this repository
2. Open `index.html` in your browser
3. Or deploy to Vercel/Netlify for a live site

## Files
- `index.html` - Main HTML file
- `assets/` - CSS, JavaScript, and image files
